Objective
The objective of this assignment is to understand and apply recursion in Java by solving problems related to numbers, sequences, and strings.
All tasks were implemented using recursion without using any loops.

Summary of Work
In this assignment, I implemented 10 different recursive functions.
Each problem was solved by defining:
•	A base case to stop recursion
•	A recursive step to reduce the problem size
The tasks include working with numbers (digits, factorial, prime), sequences (Fibonacci, power, reverse order), and strings (digit check, length).
The program was tested using sample inputs, and correct outputs were obtained for all tasks.
